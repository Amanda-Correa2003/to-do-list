📝 To-Do List Moderna

Uma aplicação web de lista de tarefas desenvolvida com React, TypeScript e Tailwind CSS, apresentando um design moderno e elegante inspirado no modo escuro.


🚀 Funcionalidades

- Adicionar Tarefas: Insira novas tarefas facilmente através do campo de texto.
- Marcar como Concluída: Conclua ou desmarque tarefas com feedback visual imediato (texto tachado e opacidade reduzida).
- Excluir Tarefas: Remova itens indesejados da lista de forma rápida.
- Interface Responsiva & Dark Mode: Estilo moderno com paleta de cores escura e acabamento polido.


🛠️ Tecnologias Utilizadas

- React (com TypeScript)
- Vite (Empacotador e servidor de desenvolvimento)
- Tailwind CSS (Estilização)

Certifique-se de ter o [Node.js](https://nodejs.org/) instalado em sua máquina.
